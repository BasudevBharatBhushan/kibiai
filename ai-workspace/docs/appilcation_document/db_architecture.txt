KIBIAI — DATABASE ARCHITECTURE
================================================================================

All PKs are UUID. All entities carry company_id for tenant isolation.
All timestamps are timestamptz.

--------------------------------------------------------------------------------
companies
--------------------------------------------------------------------------------
company_id        uuid          PK
company_name      varchar(150)
company_logo      text          URL
company_address   text
license_key       varchar(100)  Unique per tenant
plan_code         varchar(50)
status            varchar(20)   Active / Inactive / Suspended
created_on        timestamptz
updated_on        timestamptz

--------------------------------------------------------------------------------
users
--------------------------------------------------------------------------------
user_id           uuid          PK
auth_user_id      uuid          Supabase auth reference
company_id        uuid          FK -> companies
user_email        varchar(150)
full_name         varchar(150)
designation       varchar(120)  Job title
user_status       varchar(20)   Active / Invited / Disabled
created_on        timestamptz
updated_on        timestamptz

--------------------------------------------------------------------------------
roles
--------------------------------------------------------------------------------
role_id           uuid          PK
company_id        uuid          FK -> companies
role_name         varchar(80)
is_super_admin    boolean       True = full override, bypasses all permission checks
created_on        timestamptz

--------------------------------------------------------------------------------
modules
--------------------------------------------------------------------------------
module_id         uuid          PK
company_id        uuid          FK -> companies
module_name       varchar(120)  e.g. Sales, CRM, Inventory, Finance, HR
module_code       varchar(60)   Internal reference code
module_status     varchar(20)   Active / Archived
created_on        timestamptz

--------------------------------------------------------------------------------
report_templates
--------------------------------------------------------------------------------
report_template_id          uuid          PK
company_id                  uuid          FK -> companies
module_id                   uuid          FK -> modules
report_template_name        varchar(180)
conversation_id             varchar(120)  AI session reference
report_template_setup_json  jsonb         DB schema + connection config. AI reads this per prompt.
report_template_config_json jsonb         Report logic generated by AI. Versioned.
report_template_data_json   jsonb         Cached preview dataset for builder display.
report_template_insight     text          AI-generated insight for this template.
report_template_status      varchar(20)   Draft / Active / Archived
version_number              integer       Auto-incremented on each save.
created_by_user_id          uuid          FK -> users
created_on                  timestamptz
updated_on                  timestamptz

--------------------------------------------------------------------------------
reports
--------------------------------------------------------------------------------
report_id               uuid          PK
company_id              uuid          FK -> companies
report_template_id      uuid          FK -> report_templates
report_name             varchar(180)
report_config_json      jsonb         Runtime config snapshot including applied filters.
report_data_json        jsonb         Final dataset at time of generation. Immutable.
report_insight          text          AI narrative summary of findings.
generated_by_user_id    uuid          FK -> users
created_on              timestamptz   Generation timestamp.

--------------------------------------------------------------------------------
chart_templates
--------------------------------------------------------------------------------
chart_template_id           uuid          PK
company_id                  uuid          FK -> companies
report_template_id          uuid          FK -> report_templates
conversation_id             varchar(120)  AI session reference
chart_template_name         varchar(180)
chart_template_type         varchar(40)   Bar / Line / Pie / Donut / Area
chart_template_setup_json   jsonb         Axis, series, aggregation configuration.
chart_template_dataset_json jsonb         Dataset schema mapping for rendering.
chart_template_canvas_state jsonb         Dashboard layout state.
chart_template_status       varchar(20)   Draft / Active / Archived
version_number              integer       Auto-incremented on save.
created_on                  timestamptz
updated_on                  timestamptz

--------------------------------------------------------------------------------
charts
--------------------------------------------------------------------------------
chart_id                uuid          PK
company_id              uuid          FK -> companies
chart_template_id       uuid          FK -> chart_templates
report_id               uuid          FK -> reports. Specific report this chart was built from.
chart_name              varchar(180)
chart_type              varchar(40)   Output type at generation time.
chart_json              jsonb         Final rendered chart configuration.
duplicate_of_chart_id   uuid          Nullable. References source chart if cloned.
created_by_user_id      uuid          FK -> users
created_on              timestamptz

--------------------------------------------------------------------------------
ai_usage_logs
--------------------------------------------------------------------------------
ai_usage_log_id   uuid          PK
company_id        uuid          FK -> companies
user_id           uuid          FK -> users
entity_type       varchar(30)   Report / Chart / Template
entity_id         uuid          ID of the entity being worked on.
model_name        varchar(80)   AI model used.
input_tokens      integer       Prompt token count.
output_tokens     integer       Response token count.
created_on        timestamptz

================================================================================
RELATIONSHIPS SUMMARY
================================================================================

companies
  -> users (company_id)
  -> roles (company_id)
  -> modules (company_id)
  -> report_templates (company_id)
  -> reports (company_id)
  -> chart_templates (company_id)
  -> charts (company_id)
  -> ai_usage_logs (company_id)

modules
  -> report_templates (module_id)

report_templates
  -> reports (report_template_id)
  -> chart_templates (report_template_id)

reports
  -> charts (report_id)

chart_templates
  -> charts (chart_template_id)

users
  -> report_templates (created_by_user_id)
  -> reports (generated_by_user_id)
  -> charts (created_by_user_id)
  -> ai_usage_logs (user_id)
